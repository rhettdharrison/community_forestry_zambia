CFMG Equitable Benefit Sharing

Author

Rhett D Harrison

Introduction

Here we consider the equitable sharing of benefits arising from community forest management (CFM) among CFMG members.

Benefit sharing

We asked FGD groups whether the benefits of CFM were equitably distributed (Figure 1).

Code 
cfmg |>
  filter(interview_type != "individual_interview") |>
  mutate(
    response = economic_transformation_benefits_equity_across_groups,
    response = case_match(response,
                          "yes" ~ "Yes",
                          "no" ~ "No",
                          .default = NA),
    response = factor(response, levels = c("Yes", "No")),
    interview_type = factor(
      case_match(
        interview_type,
        "cfmg_executive" ~ "Executive",
        "cfmg_female"    ~ "Female",
        "cfmg_male"      ~ "Male",
        "cfmg_youth"     ~ "Youth"
      ),
      levels = c("Youth", "Male", "Female", "Executive")
    )
  ) |>
  drop_na(response) |>
  count(interview_type, response) |>

  mutate(prop = n / sum(n)) |>
  ungroup() |>
  ggplot(aes(y = interview_type, x = prop, fill = response)) +
  geom_col(width = 0.65, colour = "white", linewidth = 0.4) +
  geom_text(
    aes(label = ifelse(prop >= 0.05, n, "")),

NoteEquity - FGDs - Model methods

To examine differences among FGD types in whether they agreed the benefits of CFM were equitably shared, we employed a generalised linear mixed model (GLMM) with a binomial response and random intercepts for province and CFMG nested within province. We used FGD type as the factor. We examined the significance of pairwise differences among FGD types using Tukey HSD.

Code 
test.data <- cfmg |>
  filter(interview_type != "individual_interview") |>
  mutate(
    empl = economic_transformation_benefits_equity_across_groups,
    empl = case_match(empl,
                      "yes" ~ 1,
                      "no" ~ 0,
                      .default = NA),
    interview_type = factor(
      case_match(
        interview_type,
        "cfmg_executive" ~ "Executive",
        "cfmg_female"    ~ "Female",
        "cfmg_male"      ~ "Male",
        "cfmg_youth"     ~ "Youth"
      ),
      levels = c("Executive", "Female", "Male", "Youth")
    )
  ) |>
  drop_na(empl) |>
  mutate(province_cfmg = interaction(province, cfmg_name))

M1 <- glmer(empl ~ interview_type + (1 | province) + (1 | province_cfmg),
            data = test.data, family = binomial)

emm.M1      <- emmeans(M1, specs = ~ interview_type, type = "response")
pairwise.M1 <- pairs(emm.M1, adjust = "Tukey")

Individual interviewees were also asked whether the benefits of CFM were equitably shared. Table 1 shows the percentage reporting this, broken down by interviewee role.

Code 
cfmg |>
  filter(interview_type == "individual_interview") |>
  select(
    interviewee_role,
    empl = economic_transformation_benefits_equity_across_groups
  ) |>
  mutate(
    interviewee_role = case_match(
      as.character(interviewee_role),
      "cfmg_member"      ~ "Community member",
      "cfmg_exec_member" ~ "CFMG executive member",
      "trad_leader"      ~ "Traditional leader",
      "dfo"              ~ "District Forestry Officer",
      "NGO_rep"          ~ "NGO representative"
    ),
    interviewee_role = factor(interviewee_role, levels = c(
      "Community member", "CFMG executive member", "Traditional leader",
      "District Forestry Officer", "NGO representative"
    )),
    empl = case_match(empl,
                      "yes" ~ 1,
                      "no" ~ 0,
                      .default = NA)
    
  ) |>
  group_by(interviewee_role) |>
  summarise(
    n          = n(),
    Equity = mean(empl, na.rm = TRUE),
    .groups    = "drop"
  ) |>
  arrange(interviewee_role) |>
  gt() |>
  cols_label(
    interviewee_role = "Role",

  ) |>
  fmt_percent(
    columns  = Equity,
    decimals = 0
  ) |>
  tab_header(
    title    = "Equitable sharing of benefits, as reported by individual interviewees",

  ) |>
  opt_align_table_header(align = "left") |>
  cols_align(align = "left",   columns = interviewee_role) |>
  cols_align(align = "center", columns = c(n, Equity)) |>
  tab_style(
    style     = cell_text(align = "center"),
    locations = cells_column_labels(columns = interviewee_role)
  ) |>
  sub_missing()
NoteEquity - Individual interviewees - Model methods

To examine differences among individual interviewee roles in whether they agreed the benefits of CFM were equitably shared, we employed a generalised linear mixed model (GLMM) with a binomial response and random intercepts for province and CFMG nested within province. We included interviewee role, interviewee gender and interviewee age as factors. To examine pairwise differences among interviewee roles, we employed Tukey HSD.

Carbon income benefit sharing

Company-community arrangements

Executive FGDs reported which elements were included in their carbon finance benefit-sharing arrangement (Figure 2).

Code 
prefix_bs <- "economic_transformation_benefit_sharing_arrangement_company_community_"

bs_labels <- c(
  "revenue_sharing_percentage_of_carbon_finance_proceeds_given_to_the_community"              = "Revenue sharing",
  "employment_opportunities_for_community_members"                                             = "Employment",
  "investment_in_community_infrastructure_e_g_schools_clinics_roads"                          = "Infrastructure investment",
  "support_for_sustainable_livelihoods_e_g_alternative_income_generating_activities_training_programs" = "Livelihood support"
)

full_bs_vars <- paste0(prefix_bs, names(bs_labels))

cfmg |>
  filter(interview_type == "cfmg_executive") |>
  select(all_of(full_bs_vars)) |>
  summarise(across(everything(), ~ sum(. == 1, na.rm = TRUE))) |>
  pivot_longer(everything(), names_to = "variable", values_to = "n") |>
  mutate(label = factor(bs_labels[sub(prefix_bs, "", variable, fixed = TRUE)],
                        levels = rev(bs_labels))) |>
  ggplot(aes(x = n, y = label)) +
  geom_col(fill = "#2A9D8F", width = 0.55) +
  geom_text(aes(label = n), hjust = -0.3, size = 5, colour = "grey30") +
  scale_x_continuous(
    expand = expansion(mult = c(0, 0.15)),
    name   = "Number of CFMGs"
  ) +
  scale_y_discrete(expand = expansion(add = 0.5)) +
  labs(y = NULL) +
  theme_minimal(base_size = 18) +
  theme(
    panel.grid.major.y = element_blank(),
    panel.grid.minor   = element_blank(),
    panel.grid.major.x = element_blank(),
    axis.line.x.bottom = element_line(colour = "grey30", linewidth = 0.5),
    axis.text          = element_text(colour = "grey20"),
    plot.margin        = margin(10, 25, 10, 10)
  )
