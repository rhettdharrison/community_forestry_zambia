CFMG Sustainable Forest Management

Author

Rhett D Harrison

Introduction

Here we consider the effect of CFMG management on indicators of sustainable forest management.

Deforestation

We asked interviewees whether community forest management has reduced deforestation inside the community forest, and whether it has increased outside the community forest (leakage).

Deforestation inside the community forest

As can be seen, a majority of groups report a moderate or significant reduction in deforestation in the community forest. Below we examine whether these differences among FDG categories are significant. However, the results indicate that the differences in responses among FDG groups were not significant.

NoteDeforestation - FGDs - Model methods

To examine the differences among FGD types in their assessment of whether CFM had reduced forest clearing, we employed a cumulative link mixed model (CLMM) with random intercepts for province and CFMG nested within province. We used FGD type as the factor. We examined the significance of pairwise differences among FGD types using Tukey HSD.

Code 
test.data <- cfmg |>
  filter(interview_type != "individual_interview") |>
  mutate(
    deforestation = case_match(
      impact_on_sfm_cfm_reduced_forest_clearing_cfma,
      "no_there_has_never_been_clearing_for_agriculture" ~ "No change",
      "Agricultural practices ceased way before CFM"     ~ "No change",
      "One field for cassava"                            ~ "No change",
      "Yes there no more fields_  it's all forests"      ~ "Significant",
      "Yes_ significant reduction. There was chitemene system before CFM" ~ "Significant",
      "no_increase"    ~ "No change",
      "yes_moderate"   ~ "Moderate",
      "yes_significant" ~ "Significant",
      .default = NA
    ),
    deforestation = factor(deforestation,
                           levels = c("No change", "Moderate", "Significant"),
                           ordered = TRUE),
    interview_type = factor(
      case_match(
        interview_type,
        "cfmg_executive" ~ "Executive",
        "cfmg_female"    ~ "Female",
        "cfmg_male"      ~ "Male",
        "cfmg_youth"     ~ "Youth"
      ),
      levels = c("Executive", "Female", "Male", "Youth")
    )
  ) |>
  drop_na(deforestation) |>
  mutate(province_cfmg = interaction(province, cfmg_name))

M1 <- clmm(deforestation ~
              interview_type +
              (1 | province) +
              (1 | province_cfmg),
            data = test.data)

emm.M1     <- emmeans(M1, specs = ~ interview_type)
pairwise.M1 <- pairs(emm.M1, adjust = "Tukey")
